# 바른자세 — 실시간 자세 교정 앱

웹캠 기반 실시간 자세 분석 및 교정 알림 애플리케이션입니다.
MediaPipe Pose를 활용해 목 굴곡, 머리 앞돌출, 측방 기울기, 어깨 기울기를 측정하고
자체 개발 채점 기준인 **PSI(Posture Score Index)** 로 점수를 산출합니다.

---

## 주요 기능

- 실시간 웹캠 자세 분석 (30fps)
- PSI 18점 만점 채점 — 낮을수록 좋은 자세
- 5초 기준 설정으로 개인별 기준 설정
- 자세 등급에 따른 팝업 알림 (주의 / 경고 / 위험)
- 일별 캘린더 + 주간/월간 통계 차트
- 리포트 페이지: 기간별 평균 PSI, 시간대별 패턴, 등급 분포, 스트레칭 권장
- 히스토리 페이지: 날짜별 기록 조회
- 알림 간격 설정

---

## PSI 채점 기준

PSI는 4개 축을 독립 채점 후 가중 합산하는 방식입니다.

| 축 | 측정 항목 | 가중치 | 점수 범위 |
|---|---|---|---|
| 축1 | 목 전방 굴곡 (도) | x2 | 1~3 |
| 축2 | 머리 앞돌출 (눈 너비 비율) | x1 | 1~4 |
| 축3 | 측방 기울기 (도) | x1 | 1~4 |
| 축4 | 어깨 기울어짐 (도) | x1 | 1~4 |

**총점 = (축1 x 2) + 축2 + 축3 + 축4 — 범위: 5~18**

| PSI 점수 | 등급 | 의미 |
|---|---|---|
| 5 | 완벽 | 이상적인 자세 |
| 6 ~ 8 | 허용 | 양호한 자세 |
| 9 ~ 12 | 주의 | 자세 교정 권장 |
| 13 ~ 15 | 경고 | 즉시 교정 필요 |
| 16 ~ 18 | 위험 | 거북목 위험 |

## 측정 방식

- **목 굴곡**: world landmarks 기준 눈 중점→입 중점 벡터의 Y-Z 평면 atan2 각도 — 기준 설정값 차이 (얼굴 피치 직접 측정)
- **머리 앞돌출**: 양쪽 눈 사이 픽셀 거리 비율 — 기준 설정 대비 증가율 (핀홀 카메라 원리, MediaPipe Z값 미사용)
- **측방 기울기**: 눈~눈 선의 atan2 각도 (수평 대비 기울어진 각도)
- **어깨 기울기**: 양쪽 어깨 y좌표 차이의 atan2 각도
- 모든 측정값은 deque(maxlen=20) 이동 평균으로 스무딩

## 성능 최적화

- **빠른 모델 초기화**: MediaPipe `model_complexity=0` 적용 — 앉은 자세 측정에 충분한 정확도를 유지하면서 초기화 속도 2~3배 향상
- **앱 시작 시 사전 로딩**: 앱 실행과 동시에 백그라운드에서 AI 모델을 미리 로드 — 측정 버튼을 누를 때 대기 시간 단축
- **세션 간 모델 재활용**: 측정을 종료했다 다시 시작할 때 기존 모델 인스턴스를 재사용 — 두 번째 측정부터 즉시 시작

**요구사항**: Python 3.10, 웹캠

```bash
pip install -r requirements.txt
py posture.py
```

## 사용법

1. 앱 실행 후 웹캠 앞에 바르게 앉습니다.
2. 대시보드에서 **모니터링 시작** 버튼을 누릅니다.
3. **기준 설정** (5초): 현재 자세를 기준값으로 설정합니다.
4. 기준 설정 완료 후 카메라 창이 자동으로 숨겨지고 백그라운드 측정이 시작됩니다.
5. 자세가 나빠지면 화면에 경고 배너가 표시됩니다.
6. 카메라 창의 **기준 재설정** 버튼으로 기준 자세를 재설정할 수 있습니다.

## 데이터 저장 위치

측정 데이터는 JSON 파일로 자동 저장됩니다.

- Windows: `C:\Users\[사용자명]\.posture_app\posture_data.json`
- Mac / Linux: `/home/[사용자명]/.posture_app/posture_data.json`

앱을 처음 실행하면 자동으로 파일과 폴더가 생성됩니다.
다른 기기에서 데이터를 가져오려면 해당 경로에 JSON 파일을 복사하면 됩니다.
