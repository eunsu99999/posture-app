# Posture App

실시간 웹캠 기반 자세 교정 앱입니다. MediaPipe Pose를 사용해 목 굴곡각도, 앞돌출, 좌우 기울기를 측정하고 RULA 기준으로 자세를 채점합니다.

## 주요 기능

- **실시간 자세 분석**: 웹캠으로 목 굴곡각도, 앞돌출(거북목), 좌우 기울기를 실시간 측정
- **RULA 기반 채점**: 국제 표준 RULA(Rapid Upper Limb Assessment) 목 점수 기준 1~5점 채점
- **캘리브레이션**: 사용자 체형·카메라 위치에 맞게 개인화된 기준점 설정
- **경고 알림**: 자세가 나빠지면 화면 상단에 팝업 배너 표시 (3점 이상)
- **히스토리**: 날짜별 자세 기록 캘린더 및 상세 통계
- **대시보드**: 오늘의 자세 점수, 세션 시간, 스트레칭 목표 관리
- **리포트**: 주간·월간 자세 분석 차트

## RULA 채점 기준

| 점수 | 등급 | 기준 |
|------|------|------|
| 1점 | 완벽 | 굴곡 0~10° |
| 2점 | 허용 | 굴곡 10~20° |
| 3점 | 주의 | 굴곡 20° 초과 |
| 4점 | 경고 | 후신전(뒤로 젖힘) |
| 5점 | 위험 | 위 조건 + 좌우 기울기 초과 |

> 굴곡각도는 수직 픽셀 거리 및 MediaPipe 3D 월드좌표 z축을 조합해 추정합니다.

## 설치

### 요구사항

- Python 3.10
- 웹캠

### 패키지 설치

```bash
pip install -r requirements.txt
```

### 실행

```bash
py -3.10 posture.py
```

## 사용법

1. 앱 실행 후 웹캠 앞에 바르게 앉습니다.
2. **캘리브레이션** (5초): 현재 자세를 기준값으로 설정합니다.
3. 모니터링이 시작되면 카메라 창이 자동으로 숨겨집니다.
4. 자세가 나빠지면 화면 우상단에 경고 배너가 표시됩니다.
5. **다시 캘리브레이션** 버튼으로 기준 자세를 재설정할 수 있습니다.

## 민감도 설정

| 단계 | 좌우 기울기 임계값 |
|------|------------------|
| 편안하게 | 15° 초과 시 +1점 |
| 균형잡힌 (기본) | 10° 초과 시 +1점 (RULA 표준) |
| 엄격하게 | 7° 초과 시 +1점 |

## 데이터 저장 위치

사용자 데이터는 아래 경로에 자동 저장됩니다.

- Windows: `C:\Users\사용자명\.posture_app\posture_data.json`
- Mac/Linux: `/home/사용자명/.posture_app/posture_data.json`
